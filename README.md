# 🔒 Synthetic Data Generation & Evaluation

## 📌 Overview
This project generates synthetic datasets and evaluates their quality using statistical, machine learning, and privacy metrics.

---

## 🚀 Features
- Synthetic data generation using **Gaussian Noise & Gaussian Copula**  
- Statistical similarity evaluation using **KS Test**  
- ML utility evaluation using **TSTR (Train on Synthetic, Test on Real)**  
- Privacy evaluation using **distance-based metrics**  
- Unified scoring combining all evaluation metrics  
- Sensitivity analysis across different use cases  

---

## 🛠️ Tech Stack
Python | Pandas | NumPy | Scikit-learn | Copulas
